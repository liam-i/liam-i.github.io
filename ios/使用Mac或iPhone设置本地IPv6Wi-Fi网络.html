# 使用 Mac 设置本地 IPv6 Wi-Fi 网络

操作步骤请查看 Apple 开发者网站：To set up a local IPv6 Wi-Fi network using your Mac在新窗口打开

# 使用 iPhone 设置本地 IPv6 Wi-Fi 网络

# 背景

使用 Mac 设置本地 IPv6 Wi-Fi 网络要求 Mac 必须带有 RJ45端口，但 Apple 新款 MacBook 都已取消了 RJ45 端口，所以我选择使用 iPhone 设置本地 IPv6 Wi-Fi 网络。

# 所需设备

  • iPhone 一台（蜂窝网络可用）
  • MacBook 一台
  • Lightning 线一根
  • 其他设备一台（iPhone、Android、Mac、Windows 等等）

# 操作步骤

  1. 用 Lightning 线将 iPhone 与 Mac 相连

  2. 打开 Mac 的网络偏好设置，确保 Wi-Fi 是打开的，并且没有连接到任何网络

  3. 关闭 iPhone 的 Wi-Fi，打开“蜂窝网络”和“个人热点”

    提示

    如果在打开“个人热点”后，弹出以下弹窗，请选择“仅 USB”

  4. 成功打开 iPhone 的“个人热点”之后，确保 Mac 的网络已连接“iPhone USB”

  5. 打开 Mac 的系统偏好设置，先按住 option 键，再点击共享（点击共享时按住 option 键不能松开）

  6. 在“服务”中选择“互联网共享”，然后在“共享以下来源的连接”中选择“iPhone USB”，在“用以下端口共享给电脑”中勾选“Wi-Fi”，最后勾选“创建NAT64网络”

  7. 在“服务”中勾选“互联网共享”，然后“配置共享互联网的网络”，配置完成点击按钮“好”，这时我们的网络已共享完成

  8. 完成“配置共享互联网的网络”之后，确保成功勾选“互联网共享”

    提示

    到此为止“使用 iPhone 设置本地 IPv6 Wi-Fi 网络”已经设置完成
